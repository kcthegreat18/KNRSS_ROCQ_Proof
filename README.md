This ROCQ file contains the formal mechanization of the Kol Naor Rational Secret Sharing Scheme using ROCQ,  an open-source, formal proof management system, or interactive theorem prover, designed to develop mathematical proofs and verify software/hardware models.

This is in partial fulfillment for CS 199 as part of the undergraduate thesis of Keanu Abriol and Marc Dela Paz. The project is under supervision of Dr. Henry Adorna and Sir Alfonso Labao, the laboratory heads of the Logic and Computability Lab of the Department of Computer Science, UP Diliman.
